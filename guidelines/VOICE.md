# Rituales · Voz y personalidad

Este documento define cómo escribe, habla y piensa Rituales.
Es una guía viva — se actualiza cada vez que una decisión editorial cambia algo.

---

## Propósito

Rituales es una herramienta de autoconocimiento.

El ritual es el medio. Conocerse mejor para decidir y actuar con más claridad es el fin.
No es una app de bienestar genérica. No es meditación como escape.
Es un espacio para detenerse, observar sin juicio, y desde ahí tomar mejores decisiones.

---

## Tono

**Contemplativo** — significa actuar con presencia, observación y sin juicio.
Es el punto de partida para tomar decisiones, no un estado de aceptación resignada.
No romantiza la pérdida, la impermanencia ni el soltar.
Todo el contenido tiene enfoque positivista anclado a la realidad: orientado a la acción consciente.

**Directo** — sin rodeos, sin eufemismos new age vacíos.
Si algo se puede decir en menos palabras, se dice en menos palabras.

**Encarnado** — habla del cuerpo, la respiración, los sentidos.
La práctica es física. El lenguaje también.

**Rioplatense** — tuteo con vos, naturalidad porteña.
Nunca neutro. Nunca español de España.
`respirá`, `sentís`, `acomodate`, `decile`, `quedás`.

---

## Reglas de escritura

### Sin género gramatical
Los textos no usan `ella` ni `él`.
El género queda solo en los sustantivos cuando aplica naturalmente.
Se usa segunda persona (`vos`, `tu`, `te`) o construcciones impersonales.

### Sin tecnología
La herramienta no existe en el copy. Solo la práctica.
Nunca: `IA`, `inteligencia artificial`, `algoritmo`, `modelo`, `sistema`.
Sí: `diez minutos`, `tu voz`, `una intención`.

### Sin new age vacío
Nunca: `energía cósmica`, `vibración`, `manifestar`, `universo` (como agente), `sanar`, `trabajar emocionalmente`.
El lenguaje espiritual se usa cuando es concreto y específico, no como decoración.

### Sin resignación
Nunca frases que romantizen la pérdida, la impermanencia o el soltar como destino.
Mal: `la plenitud no dura, por eso vale tanto`.
Bien: `lo que construiste este mes ya existe en el mundo`.

### Ritmo fragmentado
Frases cortas. Comandos directos. Puntos solos.
`Respirá. Sentí el peso de tus pies. Eso es todo por ahora.`
No párrafos largos explicativos.

### La pregunta como herramienta central
El contenido abre en vez de cerrar.
Una pregunta bien hecha vale más que diez respuestas.
`¿Qué ves si te quedás quieto?` — no `aquí te explicamos cómo encontrar claridad`.

## Palabras prohibidas

| Nunca usar | Por qué |
|---|---|
| `sanar` | Clínico y vago |
| `terapia` | Fuera de alcance |
| `emoción` (en sentido clínico) | Rompe el tono |
| `positivo / negativo` | Juicio innecesario |
| `problema` | Encuadre errado |
| `trabajar` (lo emocional) | New age vacío |
| `relajate` | Cliché |
| `respira profundo` | Cliché |
| `suelta el estrés` | Cliché |
| `estás en un lugar seguro` | Cliché |
| `luz blanca` | Cliché de meditación genérica |
| `ella` / `él` | Género gramatical — no se usa |
| `IA`, `inteligencia artificial` | La tecnología no aparece |

### Instrucción de ritual (apertura)
```
✓ "Buscá un lugar tranquilo. Apagá notificaciones. Cerrá los ojos y tomá cuatro respiraciones profundas, soltando tensión con cada exhalación."
✗ "Para comenzar este poderoso ritual de sanación, lo que necesitamos hacer es encontrar un espacio sagrado donde podamos conectar con nuestra energía."
```

### Afirmación de cierre
```
✓ "Ya tengo la respuesta que necesito — solo necesito escucharla."
✓ "Puedo con esto — y también puedo descansar."
✗ "Soy poderosa, merezco lo mejor y el universo me respalda."
```

### Micro-copy de story
```
✓ "No hay nada que resolver ahora mismo. Este ritual de 5 minutos es un permiso para dejar de correr."
✓ "Reconectar con vos no es un lujo — es el punto de partida de todo lo demás. Este ritual dura 10 minutos y cuesta cero."
✗ "Realizá este ritual holístico de amor propio para elevar tu vibración y sanar tus heridas más profundas."
```

### Placeholder de intención
```
✓ "Quiero tener claridad para tomar una decisión que lleva tiempo rondándome."
```

## Formato de UI copy

Decisiones de estilo para elementos pequeños que se repiten en toda la app.

| Elemento | Formato | Ejemplo |
|---|---|---|
| Eyebrow / etiqueta | Mayúscula total | `RITUAL DE CLARIDAD` |
| CTA principal de ritual | Siempre igual | `Hacer este ritual` |
| CTA de detalle | Siempre igual | `Ver ritual` |
| Separador de metadatos | Espacio-punto-espacio | `Agua · 10 min` |
| Duración | Sin abreviación | `5 minutos` en body, `10 min` en metadata |
| Descripción de elemento | Dual-noun con punto medio | `Cuerpo · Estabilidad` |
| Signos de admiración | Solo en feedback de sistema | `Ritual guardado ✓` |

---

## Elementos y su cualidad

Los cuatro elementos son el lenguaje de los estados internos.
No se usan como decoración — cada uno propone una práctica concreta.

| Elemento | Cualidad | Para cuando |
|---|---|---|
| Agua | Claridad, fluir | Demasiada información, ninguna certeza |
| Tierra | Calma, arraigo | Todo se mueve demasiado rápido |
| Fuego | Enfoque, transformación | Energía dispersa, necesidad de dirección |
| Aire | Soltar, perspectiva | Carga que no es propia, mente circular |

## Estructura del ritual

El ritual tiene cinco secciones. El ritmo del texto se vuelve más lento hacia el núcleo.

Marcas de pausa para TTS:
- `[PAUSA_CORTA]` — 2–3 segundos
- `[PAUSA_MEDIA]` — 4–6 segundos
- `[PAUSA_LARGA]` — 8–12 segundos

---

## Base teórica (no se menciona explícitamente en el contenido)

- **Teoría Polivagal (Porges)** — el ritual como regulación fisiológica, no psicológica
- **Rueda de las emociones (Plutchik)** — para mapear estado emocional a tipo de práctica
- **Ventana de tolerancia (Siegel)** — el ritual opera dentro de la ventana, no la fuerza
- **Principio de los elementos** — agua, tierra, fuego, aire como antídotos específicos a estados del sistema nervioso

## Diario de decisiones

Registro de los cambios de criterio que fueron afinando la voz.

### Abril 2026 — Sin género gramatical
**Decisión:** los textos no usan `ella` ni `él`. El género queda solo en sustantivos.
**Por qué:** el texto funciona para cualquier persona sin esfuerzo. No es solo inclusión — es que la segunda persona (`vos`, `te`) ya es el registro natural de la app.

### Abril 2026 — Contemplativo ≠ aceptación
**Decisión:** "contemplativo" se redefine. No significa aceptar lo que se va. Significa presencia + observación + sin juicio como punto de partida para tomar decisiones.
**Por qué:** frases como `la plenitud no dura, por eso vale tanto` se leían como resignación disfrazada de sabiduría. El tono tiene que ser positivista y anclado a la realidad.

### Abril 2026 — Propósito central: autoconocimiento
**Decisión:** Rituales es una herramienta de autoconocimiento. El ritual es el medio. Conocerse mejor para decidir y actuar con más claridad es el fin.
**Por qué:** "bienestar" es demasiado genérico y se confunde con el mercado de wellness. "Autoconocimiento" nombra con precisión para qué sirve la práctica.

### Abril 2026 — La tecnología no aparece en el copy
**Decisión:** ningún texto visible al usuario menciona IA, algoritmo, modelo ni tecnología.
**Por qué:** la propuesta de valor es la experiencia, no el cómo. Nombrar la tecnología rompe la presencia y desplaza la atención hacia el medio en vez del fin.

### Fecha a completar — Sin luz blanca
**Decisión:** eliminar la imagen de "luz blanca" de todos los rituales.
**Por qué:** es el cliché más gastado de la meditación genérica. Rompe la presencia y suena a app de bienestar de los 2010.
